## Requirements

1. <a href="https://nodejs.org/en/download/">Node Js</a>
2. <a href="https://reactjs.org/docs/getting-started.html">React JS<a/>


## Installation

1. Clone this Project or you can type `git clone https://github.com/audywp/Teravin-front-end.git`
2. Clone Node Project you can type `git clone https://github.com/audywp/teravin.git` or open `$ROOT_DIRECTORY/Teravin-front-end/src/utils/config` change `APP_BACKEND: 'http://localhost:3060/'` to `APP_BACKEND: 'http://3.90.42.24:3060/'` 
2. Open app's directory in CMD or Terminal
3. Type `yarn install / npm install`
4. Type ` yarn start / npm start`
